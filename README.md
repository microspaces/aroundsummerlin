# Around Summerlin — Website

Clean, Substack-style landing page for the Around Summerlin weekly newsletter.

## Structure

```
site/
├── index.html          # Main landing page
├── css/
│   └── style.css       # All styles (responsive, mobile-first)
├── js/
│   └── main.js         # Form handling, smooth scroll, animations
└── README.md           # This file
```

## Features

- **Hero section** with email capture
- **Features grid** — what subscribers get
- **Sample issue preview** — newsletter mockup
- **CTA section** — secondary subscribe prompt
- **Footer** — nav links, social, advertise
- **Success modal** — post-subscribe confirmation
- **Fully responsive** — mobile, tablet, desktop
- **Smooth animations** — fade-in, scroll effects
